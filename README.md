# 📚 下节啥课

> 一款基于 uni-app 的多课表管理应用，支持正方教务系统课表导入、多课表切换、作息时间自定义、以及 Android 常驻通知提醒。

---

## ✨ 功能特性

### 🗂 多课表管理

- 支持同时维护**多套独立课表**（如"大三下"、"大四上"），自由切换
- 每个课表独立存储课程数据、学期起始日期和作息时间
- 课表支持**新建 / 编辑 / 删除 / 切换**等完整生命周期管理

### 📥 教务系统导入

- 适配**新版正方教务系统** (`jwglxt`) 的 JSON 数据接口
- 自动解析课程名称、教师、教室、节次、周次（支持单双周）
- 支持 WebView 内嵌登录教务系统抓取数据，或者手动粘贴 JSON 导入

### 📅 课表展示

- **周视图**课表网格，支持左右滑动切换周次
- 自动高亮"今天"所在的日期列
- 课程卡片采用 **12 色和谐配色方案**，同名课程始终保持一致颜色
- 点击课程卡片可查看**课程详情弹窗**（时间 / 教室 / 教师 / 周次）
- 支持快速选择周次的弹窗式周次选择器

### ⏰ 作息时间

- 内置**冬令时**（下午 13:30 开始）和**夏令时**（下午 14:00 开始）两种预设
- 支持逐节自定义每节课的起止时间
- 统一课时长度开关，可一键设定所有课节的时长（20~60 分钟可选）

### 🔔 Android 常驻通知（APP 端）

- 利用 Android 原生 `NotificationManager` 实现**通知栏常驻提醒**
- 智能判断当前课程状态：
  - 📅 **学期未开始** → 显示距开学天数倒计时
  - 🟢 **正在上课** → 显示课程名、教室、剩余分钟
  - ⏰ **即将上课** → 显示下节课名称及时间差
  - 🎉 **今天没课** → 轻松提示
  - ✨ **今天的课上完了** → 辛苦提示
  - 🎓 **本学期课程已结束** → 精确到天级别的结课判断
- 每分钟自动轮询刷新通知内容

## 🚀 快速开始

### 环境要求

- [HBuilderX](https://www.dcloud.io/hbuilderx.html)（推荐最新版本）
- Node.js（仅在需要微信小程序开发时）
- Android 手机或模拟器（测试通知功能）

### 运行步骤

1. **克隆项目**

   ```bash
   git clone <repository-url>
   cd school-timetable
   ```

2. **使用 HBuilderX 打开项目**
   - 打开 HBuilderX → 文件 → 导入 → 从本地目录导入

3. **运行到设备**
   - **Android 手机**：连接 USB → 运行 → 运行到手机或模拟器 → 选择设备
   - **微信小程序**：运行 → 运行到小程序模拟器 → 微信开发者工具
   - **H5 浏览器**：运行 → 运行到浏览器

4. **首次使用流程**

   ```
   打开 App → 点击"去导入" → 新建课表（输入名称 + 选择开学日期 + 选择作息预设）
   → 进入导入页面 → 登录教务系统并导入课表 → 返回主页查看课表
   ```

## 📖 使用指南

### 新建课表

1. 在主页点击右上角 **≡** 进入"我的课表"页面
2. 点击底部 **+ 新建课表** 按钮
3. 填写以下信息：
   - **课表名称**：如"大三下学期"、"大四上"
   - **开学第一天**：选择学期的第一天日期（系统会自动对齐到所在周的周一）
   - **作息预设**：选择冬令时（13:30 开始下午课）或夏令时（14:00 开始下午课）
4. 点击 **创建并导入教务系统**，跳转到导入页面

### 导入课表

目前支持适配 **新版正方教务系统**（`jwglxt`），导入方式：

- **WebView 登录方式**（APP 端）：直接在应用内打开教务系统网页，登录后自动抓取课表数据
- **JSON 粘贴方式**：从浏览器开发者工具或其他渠道获取教务系统返回的 JSON 数据，粘贴导入

### 管理多课表

在"我的课表"页面可以：

- **切换课表**：点击课表卡片即可切换（正在使用的课表带有 cyan 色边框标记）
- **编辑课表**：点击 `编辑` 按钮，可修改课表名称和开学日期
- **删除课表**：点击 `删除` 按钮，确认后删除（不可恢复）

### 自定义作息时间

在主页课表左侧的节次列（显示时间的位置）点击，即可进入作息时间编辑页：

- 开启 **所有课程时长相同** 开关后，只需设置每节课的开始时间，结束时间自动计算
- 关闭该开关后，可以分别手动设置每节课的起止时间
- 课时长度支持 **20~60 分钟** 自定义

## 🗄 数据结构

### Profile（课表档案）

```javascript
{
  id: "m1abc2def",           // 唯一ID (时间戳+随机数)
  name: "大三下学期",          // 课表名称
  settings: {
    totalWeeks: 20,          // 学期总周数
    nodesPerDay: 12,         // 每天节次数
    semesterStart: "2026-02-23",  // 学期开始日期
    currentWeek: 1           // 当前周次
  },
  courses: [ ... ],          // 课程数组
  createAt: 1708900000000    // 创建时间戳
}
```

### Course（课程）

```javascript
{
  id: "course_1",            // 唯一ID
  name: "密码学基础",         // 课程名称
  day: 2,                    // 星期几 (1=周一, 7=周日)
  startNode: 1,              // 开始节次
  endNode: 2,                // 结束节次
  room: "K-408大数据专业实验室", // 教室
  teacher: "张三",            // 教师姓名
  weeks: [1,2,3,...,16],     // 上课周次数组
  weeksText: "1-16周"        // 原始周次文本
}
```

### TimeSchedule（作息时间表）

```javascript
[
  { start: "08:00", end: "08:40" }, // 第1节
  { start: "08:45", end: "09:25" }, // 第2节
  // ... 共12节
];
```

## 🎨 设计系统

项目采用基于 **Fresh Cyan** 主色的设计系统，配色方案如下：

| 变量                    | 色值      | 用途                     |
| ----------------------- | --------- | ------------------------ |
| `--color-primary`       | `#0891B2` | 主色（按钮、高亮、链接） |
| `--color-primary-light` | `#22D3EE` | 浅主色（装饰、渐变）     |
| `--color-primary-dark`  | `#0E7490` | 深主色（按钮渐变终止色） |
| `--color-cta`           | `#22C55E` | CTA 行动色（成功、确认） |
| `--color-bg`            | `#ECFEFF` | 全局背景色               |
| `--color-text`          | `#164E63` | 主文字色                 |
| `--color-danger`        | `#EF4444` | 危险色（删除、警告）     |

UI 风格要点：

- **毛玻璃卡片**（Glassmorphism）：`backdrop-filter: blur(16px)` + 半透明白色背景
- **微交互动画**：所有可点击元素 `.clickable` 按下时 `scale(0.96)` + 透明度变化
- **圆角体系**：`12rpx` → `20rpx` → `28rpx` → `999rpx` 四级圆角

## 🔧 核心模块说明

### `utils/storage.js` — 多 Profile 存储层

提供多课表管理的完整 CRUD API：

| 函数                               | 说明                  |
| ---------------------------------- | --------------------- |
| `getProfiles()`                    | 获取所有课表档案      |
| `createProfile(name, start)`       | 新建课表并设为激活    |
| `deleteProfile(id)`                | 删除指定课表          |
| `getActiveProfileId()`             | 获取当前激活的课表 ID |
| `setActiveProfileId(id)`           | 切换激活课表          |
| `getCourses()` / `saveCourses()`   | 读写当前课表的课程    |
| `getSettings()` / `saveSettings()` | 读写当前课表的设置    |

### `utils/time.js` — 时间计算引擎

| 函数                                | 说明                                |
| ----------------------------------- | ----------------------------------- |
| `getPresetSchedule(type)`           | 获取预设作息（`winter` / `summer`） |
| `getTimeSchedule()`                 | 读取当前课表的自定义作息            |
| `saveTimeSchedule(schedule)`        | 保存自定义作息                      |
| `calculateCurrentWeek(start, date)` | 计算指定日期属于第几周              |

### `utils/next-class.js` — 课程状态分析

根据当天课程列表和当前时间，返回以下状态之一：

| 状态类型   | 含义                       |
| ---------- | -------------------------- |
| `no_class` | 今天没有课                 |
| `ongoing`  | 正在上课（含剩余时间）     |
| `upcoming` | 下节课即将开始（含倒计时） |
| `finished` | 今天所有课已上完           |

### `utils/notification.js` — Android 原生通知

基于 `plus.android` 桥接调用：

- `showPersistentNotification(title, content, subText)` — 显示/更新常驻通知
- `hideNotification()` — 隐藏通知

通知特性：

- 低优先级（`IMPORTANCE_LOW`）：不发声、不震动
- 常驻模式：`setOngoing(true)`，不可滑动清除
- 点击跳转回 App
